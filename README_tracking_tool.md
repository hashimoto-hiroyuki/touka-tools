# 追跡データ入力・閲覧・編集 — 操作マニュアル

## アプリの目的

アンケートNo.に紐づけて、HbA1c等の追跡データ（検査値・リンク・備考など）を **登録・閲覧・編集** するツールです。

---

## 画面構成（上から順に3つのパネル）

| パネル | 役割 |
|--------|------|
| **患者検索** | アンケートNo.で患者を特定する |
| **登録済みデータ** | その患者の過去データを閲覧・編集する |
| **追跡データ入力** | 新しいデータを登録する |

---

## 基本操作

### 1. 患者を検索する

1. 「No.」欄にアンケートNo.を入力
2. 「検索」ボタンを押す
3. 患者情報（ID・名前・医療機関）が表示される
4. 「▼ アンケート内容を表示」を押すと、アンケート全回答を確認できる

※ 検索後、その患者の登録済みデータが自動で表示されます。

---

### 2. 登録済みデータを閲覧する

- 患者検索後、**登録済みデータ** パネルに過去のデータが縦方向（項目名：値）で表示されます
- 複数件ある場合は区切り線で分かれます
- URLが入力されている項目はリンクとして表示され、クリックで開けます
- 「📊 追跡データをスプレッドシートで開く」ボタンで、元のスプレッドシートを直接開けます

---

### 3. 新しいデータを登録する

1. 患者を検索済みの状態で、**追跡データ入力** パネルへ
2. **測定日** を確認（デフォルトは当日）
3. 各項目（HbA1c、電顕など）に値を入力
   - 数値の項目（単位付き）→ 数値を入力
   - テキストの項目（単位なし）→ テキストやURLも入力可
4. 必要に応じて **備考** を入力
5. 「**入力する**」ボタンを押す
6. 確認ダイアログで内容を確認し「OK」
7. 「登録完了」と表示されれば成功

---

### 4. 既存データを編集する

1. 登録済みデータの各レコード下部にある「**✏️ 編集**」ボタンを押す
2. 登録済みデータパネルが **編集モード** に切り替わる
   - オレンジ枠 + 「No.X を編集中」バナーが表示される
   - 各項目に現在の値がセットされた状態でフォームが表示される
3. 変更したい項目の値を修正
4. 「**更新する**」ボタンを押す
5. 確認ダイアログで「OK」
6. 更新完了後、自動的に閲覧モードに戻る

**キャンセル** したい場合は、バナー右の「キャンセル」ボタンを押す。

---

### 5. 記録項目を追加する

新しい検査項目を追加したい場合：

1. **追跡データ入力** パネル下部の「**＋ 記録項目を追加**」ボタンを押す
2. **項目名** を入力（例：収縮期血圧）
3. **単位** を入力（例：mmHg）※ 不要なら空欄
4. 「追加する」を押す
5. スプレッドシートに列が追加され、以降のフォームに新項目が表示される

※ 単位ありの項目は数値入力欄、単位なしの項目はテキスト入力欄（URL可）になります。

## よくある質問

| 質問 | 回答 |
|------|------|
| 検索しても「見つかりません」と出る | アンケートNo.が正しいか確認。メインのスプレッドシートにそのNo.のデータがあるか確認してください |
| 登録済みデータが表示されない | 先に患者を検索してください。検索前は「患者を検索すると登録済みデータが表示されます」と表示されます |
| URLを入力したい | 単位なしの項目であればURLをそのまま入力できます。表示時はクリック可能なリンクになります |
| 測定日がずれている | スプレッドシートの日付はUTC保存のため、表示は日本時間（JST）に自動変換しています |
| 編集中に別の患者を検索したら？ | 編集モードはそのまま残ります。先に「キャンセル」を押してから検索してください |
| 間違えて登録した | 編集ボタンで値を修正できます。行の削除はスプレッドシートで直接行ってください |
| スプレッドシートを直接見たい | 「📊 追跡データをスプレッドシートで開く」ボタンから開けます |

---

## データの保存先

- **追跡データ**: HbA1c追跡スプレッドシート（自動連携）
- **患者情報**: メインのアンケート回答スプレッドシート（参照のみ）
